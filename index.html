Projects

Connect-R AI

Using Minimax Algorithm with Alpha-Beta Pruning

The Connect-R project was originally developed in C++ from scratch; it's essentially a dynamic version of Connect Four, meaning you can have any combination of rows, columns, and win conditions. I applied the minimax algorithm to dynamically construct my AI's decision tree. I also added alpha-beta pruning to prevent the function from creating unnessasary branches and incease the AI's response time. The AI's heuristic was designed such a way that it will attempt to set up multiple paths to winning. I am currently in the process of recreating this project using a javascript based game engine called phaser.io.


Handwriting Recognition

Using an Artificial Neural Network(ANN)

There are plenty of open source libraries available that greatly simplify the process of creating ANNs. However, for this project I decided to design and create my own ANN object from scratch in C++. I created a feed-forward ANN with back propagation. In implementation, I tested my ANN's ability to recognize handwritten numerical digits between 0 and 9. The overall accuracy varied based on the amount of layers/neurons I added in my ANN and the amount of training data I presented it with.


Grid World

Reinforcement learing using SARSA(lambda) Algorithm

Similarly to the ANN project, there are plenty of libraries that handle the hard work of reinforcement learning for you. However, in order to fully appricate and understand how reinforcement learning works, I opted to implement the logic behind the sarsa(lambda) algorithm from scratch in a javascript application. With the object set to explore 20% of the time and exploit 80% of the time and the decay rate set to .5, the object learned to navigate the grid in about 6 days. The process would have been much quicker if I had not have used a GUI. Moving forward, I plan to reimplement this project in a C++ project and compare the results.
